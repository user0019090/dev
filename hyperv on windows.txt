CREATION OF VM USING POWERSHELL


Create a new VM 
New-VM -Name Meta -MemoryStartupBytes 1024MB -Generation 1 -VHDPath "C:\Virt\meta.vhd" -SwitchName "Default Switch"  
 
Start a VM 
Start-VM -Name Meta
 
Connect to the VM console 
vmconnect localhost Meta
 
Get VM status 
Get-VM -Name Meta
 
 
Create a checkpoint 
Checkpoint-VM -Name Meta -SnapshotName "Checkpoint1" 
 
Restore a checkpoint 
Restore-VMCheckpoint -VMName Meta -Name " Checkpoint1" 
 
Restart a VM 
Restart-VM -Name Meta 
 
 
Stop a VM 
Stop-VM -Name Meta 
 
Hyper-V host settings 
Get-VMHost 
 
List all VMs 
Get-VM 
 
Remove a VM 
Remove-VM -Name Meta
